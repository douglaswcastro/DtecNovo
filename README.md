# vyrra

Site institucional estático para a vyrra, criado com HTML, CSS e JavaScript sem dependências externas.

Identidade: sistemas de gestão e operação para negócios físicos e concretos (academia, salão, campo de operação). Paleta grafite/painel + latão/ferrugem/musgo/osso, tipografia Space Grotesk (display) + Inter (texto) + IBM Plex Mono (dados/labels).

## Estrutura

- `index.html`: landing page principal.
- `servicos/`: páginas dedicadas para sites, sistemas web, aplicativos e automação.
- `styles/main.css`: identidade visual, responsividade e componentes.
- `scripts/main.js`: menu mobile, navegação ativa, scroll-reveal e comportamento do formulário.
- `assets/logo-vyrra.svg`: logo com wordmark escuro (ícone de manômetro/agulha), para fundos claros.
- `assets/logo-vyrra-light.svg`: variação com wordmark claro, para fundos escuros (usada no header e footer).
- `assets/favicon.svg`: ícone de manômetro/agulha simplificado.
- `assets/hero-dtec-1280.jpg` e `assets/hero-dtec-768.jpg`: imagens da hero (nomes de arquivo legados, conteúdo genérico).
- `robots.txt` e `sitemap.xml`: arquivos básicos de SEO técnico.

## Observações de produção

Substitua `https://vyrra.com.br`, `contato@vyrra.com.br` e o telefone placeholder `(00) 00000-0000` pelos dados reais antes da publicação.

## Como abrir

Abra `index.html` diretamente no navegador.
